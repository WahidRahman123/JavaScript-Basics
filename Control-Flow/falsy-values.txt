Some values are considered as false value and some are considered as true values in javascript.
Falsy values:
    false
    0
    -0
    0n (BigInt)
    ""
    null
    undefined
    NaN

Truthy values:
    "0"       //Logical
    'false'   //Logical
    " " (Space) //Logical
    {}
    []
    function(){}
